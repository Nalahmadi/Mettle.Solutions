Command Line Execution Instructions:

1. Install Java 8:
	- Download Link: https://www.oracle.com/java/technologies/javase/javase8-archive-downloads.html#license-lightbox
	- Run the executable
	- Add the bin directory to the system path variable
	
2. Install Maven3.8.8:
	- Download Link: https://dlcdn.apache.org/maven/maven-3/3.8.8/binaries/apache-maven-3.8.8-bin.zip
	- Unzip the folder
	- Add the bin directory to the system path variable

3. Clone the repository:
	# git clone https://github.com/Nalahmadi/Mettle.Solutions.git

4. Navigate to the project directory:
	# cd  Mettle.Solutions
	
5. Run the unit tests:
	# mvn clean test

Assumption:
Arry has no nun numeric values.
